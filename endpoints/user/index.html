Collection

POST

Create a new user.

Request

URL

/user

Body

The body is a json format of a user.

{
  "email": "user@example.org",
  "password": "qwerty123",
  "name": "John Doe"
}

Response

The response is the user object.

Body

{
  "email": "user@example.org",
  "name": "John Doe",
  "created_at": "2015-02-13 16:33:31.347069",
  "updated_at": "2015-02-13 16:33:31.347069",
  "superadmin": false,
  "deleted": false,
  "networks": [ ],
  "user_id": 22,
  "_links": {
    "self": {
      "href": "http:\/\/api.sensimity.com\/user\/22"
    }
  }
}
